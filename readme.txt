MyAnimu

Organize and rate your animes!

Features:
- Rating
- Watched status
- Comments

Endpoints:
/animu
- GET all animes
- POST add an anime

/animu?status=<status>
/animu?rating=<rating>
- GET anime by search string

/animu/<id>
- GET anime by ID
- PUT update anime by ID
- DELETE anime by ID

============
Using Flask with SQLAlchemy and Marshmallow to create a RESTful API.
Simple front end using basic HTML/CSS/Javascript.
